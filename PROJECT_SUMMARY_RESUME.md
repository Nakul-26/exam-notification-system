# Project: Exam Marks & WhatsApp Notification System

## Professional Summary for Resume

**Full-Stack Developer | Exam Marks & WhatsApp Notification System**
*Developed a comprehensive school management platform to streamline exam mark entry and parent communications.*

### Key Technical Achievements:
- **Scalable Architecture:** Built a multi-tenant (college-based scoping) system using **Node.js, Express, and MongoDB**, supporting role-based access control (RBAC) for Admins and Teachers.
- **WhatsApp Integration:** Integrated **Meta WhatsApp Cloud API** to automate sending exam results to parents, featuring safeguards like monthly caps and per-recipient delivery reporting.
- **Robust Data Handling:** Implemented bulk data management via **Excel (XLSX) import/export** for students, classes, teachers, and marks, including complex validation logic.
- **Security & Performance:** Engineered a secure authentication flow using **JWT with Refresh Token rotation**, CSRF protection, and account lockout mechanisms. Integrated **Helmet** and **Express Rate Limit** for enhanced security.
- **Audit & Compliance:** Developed a comprehensive **Audit Logging** system to track all administrative actions and authentication events, with CSV export capabilities for compliance.
- **Modern Frontend:** Crafted a responsive and intuitive UI using **React 19, Vite, and TypeScript**, focusing on high-performance state management and seamless user experience.

### Tech Stack:
- **Frontend:** React 19, TypeScript, Vite, Vanilla CSS, XLSX.
- **Backend:** Node.js, Express, MongoDB (Mongoose).
- **Security:** JWT, CSRF Protection, Rate Limiting, Helmet.
- **API:** Meta WhatsApp Cloud API.
- **Tools:** Git, npm, Excel Integration.
